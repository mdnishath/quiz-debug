Difference between map(),filter(),forEach(),find() in javascript

Map: The map() method take an arry and apply a function for each element of the array then create a new array of the result. map method alwasys return a new array but never change the original array
const numbers = [1, 2, 3, 4]; const doubledNumbers = numbers.map(num => num * 2); console.log(doubledNumbers); // Output: [2, 4, 6, 8]

Filter: The filter() method take an arry and apply a function for each element of the array then create a new array of if passes the condition. filter method alwasys return a new array but never change the original array
const numbers = [1, 2, 3, 4]; const evenNumbers = numbers.filter(num => num % 2 === 0); console.log(evenNumbers); // Output: [2, 4]

ForEach: The forEach() method take an arry and apply a function for each element of the array.The forEach() function does not return anything.
const numbers = [1, 2, 3, 4]; numbers.forEach(num => console.log(num)); // Output: 1, 2, 3, 4

Find: The find() method take an arry and apply a function for each element return the first matching element.If no element found it return undefined. Never change original array
const numbers = [1, 2, 3, 4]; const evenNumber = numbers.find(num => num % 2 === 0); console.log(evenNumber); // Output: 2

What is an event loop in JavaScript?

In JavaScript, the event loop is a way that handle execution of code in a single-threaded environment JavaScript is a single-threaded language, meaning that it can only execute one task at a time. However, many tasks in JavaScript are asynchronous, meaning that they do not complete immediately and require some time to complete, such as fetching data from an API or waiting for user input. The event loop is what allows JavaScript to manage these asynchronous tasks.

chooseQuiz() explained

This code defines a function chooseQuiz that takes two parameters: an index and a given answer. The purpose of this function is to store the user's answers to a quiz and display them. First, the function checks whether the given quiz has already been answered by searching for an object in the answers array with the same id as the current quiz using the find() method. If an object is found, it means that the quiz has already been answered, so the function loops through the answers array using a for...of loop and finds the index of the object with the same id as the current quiz. Once the index is found, the function replaces the existing object with a new object that has the updated givenAns property using the splice() method. If the quiz has not been answered yet, the function pushes a new object with the quiz data and the given answer to the answers array using the push() method. Finally, the function calls the displayAnswers function with the answers array as an argument to display the user's answers on the screen. Overall, this function is used to keep track of the user's quiz answers and update them as the user progresses through the quiz.
